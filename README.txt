Игра Квадратики
Играет человек против бота. Есть клетчатое поле определенного размера (например, 36*54 если брать А4 клетчатый лист тетрадный). По противоположным углам ставится по квадрату размером 1*1. Далее игроки ходят по очереди, бросают воображаемый игральный кубик - выпадает число, это размер будущего квадрата (n*n). Этот полученный квадрат надо поставить к своей территории так, чтобы было хотя бы 1 соприкосновение по сторонам (т.е. надо чтоб хотя бы 1 квадрат старой территории коснулся стороной хотя бы 1 клетку нового поставленного квадрата). Игра продолжается, пока поставить квадрат можно. Тот кто не может поставить новую фигуру - соответственно проиграл.
